## üöÄ Live Demo Features

This is a fully functional demo showcasing a modern Point of Sale system designed specifically for Koven Kafe's three branches:
- **Bomas** (Bomas of Kenya)
- **Kahawa** (Kahawa Wendani)
- **Syokimau**

### Key Features Demonstrated

#### 1. Owner Dashboard (`/dashboard`)
- **Real-time Revenue Tracking** across all branches
- **Branch Performance Comparison** with percentage changes
- **Top Selling Items** analysis
- **Recent Transactions** log
- **Quick Insights** - Peak hours, best performers, payment method analytics
- **Auto-updating Sales** - New sales appear every 10 seconds to simulate live operations

#### 2. Cashier POS Screen (`/pos`)
- **Fast Order Entry** - Touch-optimized menu grid
- **Menu Categories** - Fish Dishes, Sides, Drinks
- **Live Order Management** - Add, remove, adjust quantities
- **Multiple Payment Methods** - Cash, M-Pesa, Card
- **Receipt Generation** with print support
- **Branch Selection** - Switch between locations

#### 3. Landing Page (`/`)
- **Hero Section** with clear value proposition
- **Feature Highlights** explaining system capabilities
- **Easy Navigation** to Dashboard and POS
